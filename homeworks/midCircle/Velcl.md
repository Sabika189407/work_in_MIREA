# Минимальная окружность для множества отрезков

## Теоретическое введение

### Задача минимальной окружности
Задача минимальной окружности заключается в нахождении окружности минимального радиуса, которая содержит заданное множество геометрических объектов. В данном проекте рассматривается случай, когда объектами являются отрезки на плоскости. Минимальная окружность должна содержать все точки всех отрезков, включая их концы и внутренние точки.

### Применение
- **Компьютерная графика**: Оптимизация вычислений при работе с ограничивающими областями.
- **Геометрическое моделирование**: Построение минимальных ограничивающих областей для сложных объектов.
- **Распознавание образов**: Анализ и обработка данных, представленных в виде точек или отрезков.
- **Робототехника**: Определение минимального пространства для движения робота.

### Алгоритм Вельцля
Для нахождения минимальной окружности используется рандомизированный алгоритм Вельцля. Этот алгоритм имеет ожидаемую линейную сложность и работает следующим образом:
1. Если точек не осталось или на границе уже 3 точки, окружность вычисляется напрямую.
2. Выбирается случайная точка `p`, и рекурсивно вычисляется минимальная окружность для оставшихся точек.
3. Если `p` не входит в эту окружность, то `p` должна находиться на границе минимальной окружности.

Алгоритм Вельцля был адаптирован для работы с отрезками, где критическими точками являются концы отрезков, их середины и точки пересечения.

## Детали реализации

### Основные этапы
1. **Обработка входных данных**:
   - Если входной вектор отрезков пуст, возвращается недопустимая окружность (радиус = -1).
   - Если задан только один отрезок, возвращается окружность, центр которой находится в середине отрезка, а радиус равен половине длины отрезка.

2. **Извлечение критических точек**:
   - Концы каждого отрезка.
   - Середины каждого отрезка.
   - Точки пересечения между всеми парами отрезков.

3. **Вычисление минимальной окружности**:
   - Используется алгоритм Вельцля для нахождения минимальной окружности, содержащей все критические точки.

4. **Проверка результата**:
   - Убедиться, что все точки всех отрезков находятся внутри окружности.

### Особенности реализации
- **Коллинеарные точки**: Если три точки коллинеарны, окружность строится по двум наиболее удаленным точкам.
- **Погрешность вычислений**: Для сравнения расстояний используется небольшая погрешность (например, `1e-8`), чтобы избежать ошибок из-за ограниченной точности чисел с плавающей запятой.

## API

### Структуры

#### `Point2D`
Представляет точку на плоскости.
```cpp
struct Point2D {
    double x, y; // Координаты точки
};
```

Методы:
- `bool operator==(const Point2D& other) const`: Проверяет равенство двух точек.
- `bool operator!=(const Point2D& other) const`: Проверяет неравенство двух точек.

#### `LineSegment`
Представляет отрезок на плоскости.
```cpp
struct LineSegment {
    Point2D start; // Начальная точка отрезка
    Point2D end;   // Конечная точка отрезка
};
```

#### `Circle`
Представляет окружность.
```cpp
struct Circle {
    Point2D center; // Центр окружности
    double radius;  // Радиус окружности
};
```

Методы:
- `bool isValid() const`: Проверяет, является ли окружность допустимой (радиус >= 0).
- `bool contains(const Point2D& point) const`: Проверяет, содержится ли точка внутри окружности.
- `bool contains(const LineSegment& segment) const`: Проверяет, содержится ли отрезок внутри окружности.

---

### Основные функции

#### `Circle MinimumEnclosingCircleForSegments(const std::vector<LineSegment>& segments)`
Находит минимальную окружность, содержащую все заданные отрезки.
- **Параметры**:
  - `segments`: Вектор отрезков.
- **Возвращает**: Минимальную окружность.

#### `double distance(const Point2D& p1, const Point2D& p2)`
Вычисляет Евклидово расстояние между двумя точками.

#### `Circle circleFromDiameter(const Point2D& p1, const Point2D& p2)`
Создает окружность по двум точкам (диаметр).

#### `Circle circleFrom3Points(const Point2D& p1, const Point2D& p2, const Point2D& p3)`
Создает окружность, проходящую через три точки.

## Использование

### Пример использования
```cpp
#include "min_circle.h"
#include <iostream>

int main() {
    // Создаем несколько отрезков
    LineSegment segment1 = {{0, 0}, {4, 0}};
    LineSegment segment2 = {{2, 0}, {2, 3}};

    // Создаем вектор отрезков
    std::vector<LineSegment> segments = {segment1, segment2};

    // Находим минимальную окружность
    Circle result = MinimumEnclosingCircleForSegments(segments);

    // Выводим результат
    std::cout << "Центр: (" << result.center.x << ", " << result.center.y << ")" << std::endl;
    std::cout << "Радиус: " << result.radius << std::endl;

    return 0;
}
